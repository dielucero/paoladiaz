# Instrucciones para agregar tus canciones MP3

Para que el reproductor funcione correctamente, necesitas agregar 3 archivos MP3 en la carpeta MUSIC:

1. cancion1.mp3 - Tu primera canción
2. cancion2.mp3 - Tu segunda canción  
3. cancion3.mp3 - Tu tercera canción

## Cómo agregar tus canciones:

1. Copia tus archivos MP3 a la carpeta MUSIC
2. Renómbralos como: cancion1.mp3, cancion2.mp3, cancion3.mp3
3. Opcionalmente, puedes cambiar los nombres en el código JavaScript

## Características del reproductor:

- ✅ Reproducción automática en loop de las 3 canciones
- ✅ Controles de play/pause, anterior, siguiente
- ✅ Barra de progreso interactiva
- ✅ Lista de canciones clickeable
- ✅ Tiempo actual y duración total
- ✅ Diseño responsive y moderno

## Personalización:

Puedes cambiar los nombres de las canciones editando el array `playlist` en el archivo JS/index.js
